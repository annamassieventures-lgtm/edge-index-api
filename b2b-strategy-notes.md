# B2B Licensing Strategy — The Edge Index

## Approach
B2B licensing is a sales conversation, not a self-serve checkout.
Do NOT add B2B tiers to Whop — it cheapens high-ticket deals.

## Flow
Landing page B2B section → "Watch How It Works" button → Proof video → Enquiry form or direct email → Close on call → Invoice via Stripe payment link

## Tiers
- Starter: $800/month (up to 500 members)
- Growth: $2,500/month (up to 5,000 members)
- Scale: $6,000/month (up to 50,000 members)
- Enterprise: Custom pricing

## What closes the deal
1. Proof the system works (screenshots, member results, accuracy data)
2. A clear ROI case (their community pays more when they trade better)
3. A satisfaction guarantee

## Automation to build later
- Calendly booking link embedded in landing page B2B section
- Post-call: automated Stripe payment link sent via email
- Onboarding sequence via MailerLite once payment confirmed
- Zapier: Stripe payment → add to Telegram group → send welcome message
